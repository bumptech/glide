class GlideSymbolProcessor(environment: SymbolProcessorEnvironment) : SymbolProcessor

Glide's KSP annotation processor.

This class recognizes and parses com.bumptech.glide.module.AppGlideModules and com.bumptech.glide.module.LibraryGlideModules that are annotated with com.bumptech.glide.annotation.GlideModule.

LibraryGlideModules are merged into indexes, or classes generated in Glide's package. When a AppGlideModule is found, we then generate Glide's configuration so that it calls the AppGlideModule and any included LibraryGlideModules. Using indexes allows us to process LibraryGlideModules in multiple rounds and/or libraries.
